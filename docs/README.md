# YCPL Documentation

YCPL is an experimental systems-programming language implemented in C++ with an
LLVM backend. This documentation describes the syntax that is intentionally
supported by the current compiler.

Start here:

- [Language Syntax](language.md)
- [Projects and Modules](projects.md)
- [Standard Library](stdlib.md)
- [Implementation Status](status.md)
- [YCPL LSP](../tools/lsp/README.md)

The rules in these documents are the source of truth for examples in this
repository. If a syntax form is not listed here, treat it as unsupported or
reserved for future work.
